# skwer

A tool for developing skwer

### Install

```
> dart pub global activate --source path tool/skwer
```

### Use

Tool documentation through the cli.

```
> skwer -h
> skwer <command> -h
```

### Develop on skwer tool

```
1. Write code in tool/skwer/lib/*.dart.
2. > skwer build-tool
3. > skwer
```
